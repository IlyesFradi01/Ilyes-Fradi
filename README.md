## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

1. **Personal Information**: Update the content in the component files:
   - `components/Hero.tsx` - Update name, title, and bio
   - `components/Skills.tsx` - Modify skills and technologies
   - `components/Projects.tsx` - Add your projects
   - `components/Experience.tsx` - Update work experience
   - `components/Contact.tsx` - Update contact information and social links

2. **Styling**: Modify `app/globals.css` and Tailwind classes in components to match your brand

3. **Colors**: Update gradient colors in components to match your preferred color scheme

## Build for Production

```bash
npm run build
npm start
```
