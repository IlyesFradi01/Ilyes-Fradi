# Project Screenshots

Place your project homepage screenshots in this folder:

- `gotogreen.png` - Screenshot of GOTOGREEN project homepage
- `sky-ascenseurs.png` - Screenshot of Sky-Ascenseurs project homepage
- `ibnsina.png` - Screenshot of IbnSina project homepage

## How to add screenshots:

1. Take a screenshot of your project's homepage
2. Save it as a PNG or JPG file
3. Name it according to the project (e.g., `gotogreen.png`)
4. Place it in this `public/projects/` folder
5. The image will automatically appear in the portfolio

## Recommended image size:
- Width: 1200px - 1920px
- Aspect ratio: 16:9 or 16:10
- Format: PNG or JPG
- File size: Keep under 500KB for best performance
